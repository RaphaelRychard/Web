ARQUITETURA BÁSICA DE UM COMPUTADOR

Para que um computador funcione são necessários alguns componentes básicos que mostraremos a seguir:

  1. Processador
  2. Memória
  3. HDD
  4. Placa Mãe
  5. Sistema Operacional

Dispositivos de Entrada e Saída (Input/Output)

É claro que você pode adicionar componentes conforme a sua necessidade, por exemplo, uma placa de vídeo, de som, ou até mesmo trocar o HD por um Pen Drive. Hoje em dia é possível fazer uma máquina funcionar sem um HD, basta fazer uso de um Live CD no Pen Drive. Sinceramente estou dizendo isso apenas como exemplo, pois o Live CD não instala o Sistema Operacional, portanto pode não ser aquilo que você procura, deixando muito a desejar.

Vamos voltar ao assunto e conhecer cada um dos componentes citados acima.

Processador

É um circuito integrado, que como seu nome diz, tem a função de processar os dados em um computador. É o componente mais complexo e normalmente o mais caro, sendo considerado o cérebro do computador.

Ele é responsável por algumas atividades, como:

  • Encontrar e executar instruções existentes na memória.
  • Realizar operações de entrada e saída.
  • Ler informações provenientes do teclado.
  • Transferir informações para monitor ou impressora.

Fabricantes mais conhecidos: AMD / Intel.

Memória RAM (Randon Access Memory ou Memória de Acesso Aleatório)

A RAM é utilizada pelo Processador para guardar (armazenar) os dados que estão sendo (processados) analisados. É uma memória volátil, pois ao se desligar o computador, todos os dados (informações) são perdidos.

Devido sua função percebemos que quanto maior a quantidade de sua memória, é possível se realizar as mais complexas atividades, resumindo, em se tratando de RAM, quanto mais, melhor.

Disco Rígido (Hard Disk ou Winchester)

Sua função é simplesmente guardar os dados, no HD ficam armazenadas todas as informações, como programas, arquivos do usuário e sistema operacional.

Placa Mãe (MotherBoard)

É responsável por conectar e interligar os componentes do computador, ou seja, fazer com que Processador, Memória RAM, Disco Rígido, entre outros fiquem interligados para que trabalhem perfeitamente.

Atualmente ouvimos muito dizer que algumas Placas Mãe são “Onboard”, isso quer dizer que a placa mãe possui um ou mais dispositivos integrados, os quais podem ser uma placa de vídeo, rede, som, etc...

Essas placas podem também contar com um chip ou capacidade de processamento de vídeos, sem ter uma placa exclusiva para determinada função.

Sistema Operacional (Operating Systems)

O Sistema Operacional é um conjunto de programas com a função de gerenciar os recursos do sistema e fornecer uma interface entre o computador e o usuário.

Exemplos: MS-DOS, Windows 8, Ubuntu, OS X.

(Android: Sistema Operacional para dispositivos móveis.)

Dispositivos de Entrada e Saída (Input/Output)

Dispositivos de Entrada: São aqueles que trazem a informação do meio externo até a memória do computador.

Exemplo: Teclado, Mouse, Microfone, Scanner, entre outros.

Teclado

Microfone

Dispositivos de Saída: São os que levam a informação da memória do computador para o meio externo.

Exemplo: Impressora, Caixa de Som, Monitor, entre outros.

Caixa de Som

Alguns outros dispositivos como Monitor Sensível a Toques, Pen Drive, Joystick Vibratório, e etc..., são considerados de Entrada e Saída, também são chamados de dispositivos Híbridos.

Pen-Drive

Voltando ao início de nosso artigo, quando me refiro a arquitetura básica de um computador, em termos de dispositivos de Entrada e Saída, basicamente me refiro a um Teclado, Monitor e Mouse, pois sem eles seria difícil interagir com a máquina.
